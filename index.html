React Posts - Dynamic Content Portal

A dynamic, role-based content management system built entirely with React. This project features a robust authentication system distinguishing between Admin and User roles, managed on the client-side using browser local storage. Admins have full CRUD (Create, Read, Update, Delete) capabilities for posts, while users have read-only access. The application also dynamically fetches and displays external content from the JSONPlaceholder REST API, demonstrating seamless integration with third-party data sources.

Expense Tracker App

A full-stack application designed to help users manage their personal finances effectively.
